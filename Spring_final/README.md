# 전자기기 쇼핑몰

Spring Boot로 개발된 전자기기 쇼핑몰 웹 애플리케이션입니다.

## 주요 기능

- 회원가입/로그인
- 상품 목록 조회
- 상품 카테고리별 분류
- 상품 정렬 (최신순, 가격순, 조회순)
- 상품 구매
- 구매 내역 조회
- 관리자 기능 (상품 추가/수정/삭제)

## 기술 스택

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Thymeleaf
- Bootstrap 5
- H2 Database

## 실행 방법

1. 프로젝트 클론
```bash
git clone https://github.com/[username]/[repository-name].git
```

2. 프로젝트 디렉토리로 이동
```bash
cd [repository-name]
```

3. 애플리케이션 실행
```bash
./mvnw spring-boot:run
```

4. 웹 브라우저에서 접속
```
http://localhost:8080
```
